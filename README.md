# Nakama Render Project

Nakama server configuration for Render deployment with Supabase PostgreSQL database.

## Environment Variables Required
- ` + "`SUPABASE_DB_HOST`" + `
- ` + "`SUPABASE_DB_USER`" + `
- ` + "`SUPABASE_DB_PASS`" + `
- ` + "`SESSION_KEY`" + `

## Deployment
This project is configured to deploy on Render.com as a Web Service.
